# 特性有効期限と有効期限(TTL)設定の設定

特性を作成する際の設定の1つは、「TTL」または「[!UICONTROL Time to Live]」の設定です。これは、指定された期間内に再資格が得られなかった場合に、特性のメンバーシップの有効期限が切れたことを意味します。 このビデオでは、この機能の表示と設定の方法を紹介します。
