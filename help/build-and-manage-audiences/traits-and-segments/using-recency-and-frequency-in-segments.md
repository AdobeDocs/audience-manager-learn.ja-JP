# 用途 [!UICONTROL Recency] および [!UICONTROL Frequency] セグメント内

用途 [!UICONTROL Recency] および [!UICONTROL Frequency] を使用すると、ある特性に対して特定の期間内に訪問者が何回資格を与えたかをセグメントパラメーターに示すことができます。 これは、コンテンツのアフィニティや頻度制限の使用例などに最適です。
