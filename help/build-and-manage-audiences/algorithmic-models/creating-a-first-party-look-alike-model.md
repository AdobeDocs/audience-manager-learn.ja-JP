# Audience Managerでの1st パーティモデルの類似の作成 {#creating-a-first-party-look-alike-model-in-aam}

このビデオでは、1st パーティの類似モデルを作成して、コンバーターのように見える独自の訪問者をより多く見つける方法を説明します。

>[!VIDEO](https://video.tv.adobe.com/v/23504/?quality=12)

これには、次のようなユースケースがあります。

* パブリッシャーは、「コンバーター」インベントリが売り切れた場合、この類似オーディエンスも販売することで拡張できます。
* コンバージョンにまだ至っていない訪問者には、サイトで具体的なマーケティングや体験を提供しましょう。

さらに、自社が所有するデータのみを使用するため、この機能は完全に無料です。 これはもちろん、あなたの利益を最大化するのに役立ちます。
