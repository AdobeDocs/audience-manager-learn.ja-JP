# [!DNL Role-Based Access Control]を使用した権限の設定

Adobe Audience Managerの[!DNL Role-Based Access Control]では、グループレベルで権限を管理でき、特性、セグメント、宛先、モデルなどのアセットの表示や操作を行うユーザーを制御できます。 権限グループを設定し、そのグループにユーザーを追加する方法を説明します。
