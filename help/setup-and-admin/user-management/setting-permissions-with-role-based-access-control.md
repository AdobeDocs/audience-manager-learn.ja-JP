# 役割ベースのアクセス制御を使用した権限の設定

Adobe Audience Managerのロールベースのアクセス制御を使用すると、権限をグループレベルで管理し、誰にアセット（特性、セグメント、宛先、モデルなど）を表示および操作させるかを制御できます。 権限グループを設定し、ユーザーを追加する方法を説明します。
