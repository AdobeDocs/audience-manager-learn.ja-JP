# [!DNL Role-Based Access Control]での権限の設定

Adobe Audience Managerの[!DNL Role-Based Access Control]を使用すると、グループレベルで権限を管理し、特性、セグメント、宛先、モデルなどのアセットの表示や操作を行うユーザーを制御できます。 権限グループを設定し、ユーザーを追加する方法について説明します。
