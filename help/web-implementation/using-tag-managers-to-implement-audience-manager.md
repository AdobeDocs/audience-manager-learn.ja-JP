# タグマネージャーを使用したAudience Managerの実装

このビデオでは、タグ管理ソリューションを使用する場合のAdobe Audience Managerの実装の基本について説明します。

主なユースケースは次のとおりです。

* Adobe Analyticsも使用する場合は、Adobe Experience Platform タグ（旧Adobe Cloud Platform Launch）を介して実装する
* Adobe Analyticsを使用しない場合のAdobe Experience Platform タグによる実装
* Adobe Analyticsを使用する場合は、Adobe以外のタグマネージャーを介して実装する
* Adobe Analyticsを使用していない場合は、Adobe以外のタグマネージャーを介して実装する
